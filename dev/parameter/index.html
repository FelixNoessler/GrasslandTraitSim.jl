Model Parameter

GrasslandTraitSim.SimulationParameter — Type

Here is an overview of the parameters that are used in the model. In addition to the parameters listed here, a regression equation with parameters not listed here from [1] is used to derive the water holding capacity and the permanent wilting point (see input_WHC_PWP!).
