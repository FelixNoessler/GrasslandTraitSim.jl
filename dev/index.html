Quick install

  1. Download Julia.

  2. Launch Julia and type

julia> using Pkg
julia> Pkg.add(url="github.com/felixnoessler/lib/GrasslandTraitValid")ERROR: Path `github.com/felixnoessler/lib/GrasslandTraitValid` does not exist.
julia> Pkg.add(url="github.com/felixnoessler/lib/GrasslandTraitVis")ERROR: Path `github.com/felixnoessler/lib/GrasslandTraitVis` does not exist.
julia> Pkg.add(url="github.com/felixnoessler/GrasslandTraitSim.jl")ERROR: Path `github.com/felixnoessler/GrasslandTraitSim.jl` does not exist.
Note

The simulations were tested with Julia 1.9.3. I recommend using this version.

Run simulations

import GrasslandTraitSim as sim
import GrasslandTraitValid as valid
mp = valid.model_parameters();
inf_p = (; zip(Symbol.(mp.names), mp.best)...);
input_obj = valid.validation_input(; plotID = "HEG01", nspecies = 25,
                                     startyear = 2009, endyear = 2021,
                                     npatches = 1);
sol = sim.solve_prob(; input_obj, inf_p);
