Getting started â€‹

GrasslandTraitSim.jl is a Julia package for simulating plant dynamics in managed grasslands.

Author: Felix NÃ¶ÃŸler.

Licence: GPL-3.0

Please contact me if you have any questions about using the model or if you would like to collaborate. You can write my an email or open an issue on Github.

Installation â€‹

  1. Download Julia.

  2. Launch Julia and type

julia
import Pkg
Pkg.add("GrasslandTraitSim")

For the tutorials you will also need several other Julia packages:

julia
Pkg.add(["CairoMakie", "GLMakie", "Unitful", "Statistics"])

Run simulations â€‹

For more details, see the tutorials on preparing inputs and analysing outputs.

julia
import GrasslandTraitSim as sim

trait_input = sim.input_traits();
nspecies = length(trait_input.amc)
input_obj = sim.validation_input(; plotID = "HEG01", nspecies);
p = sim.SimulationParameter();
sol = sim.solve_prob(; input_obj, p, trait_input);
sol.output.biomass
